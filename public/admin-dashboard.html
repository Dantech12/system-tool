Getting Started

Welcome to the Rabotec Tool Management System. This comprehensive system helps you efficiently manage tool inventory, track issuances, and generate detailed reports for audit purposes.

Real-time tool tracking
Automated reporting
Multi-shift management

Tool Management

Add Tools

Click "Add Tool" to manually add new tools with code, description, and quantity.

Import Tools from Excel

Use "Import Excel" for bulk import. Excel file must contain columns: tool_code, description, and quantity. The quantity will automatically set both total and available quantities. Existing tools will be updated with new quantities.

Example Excel format:
tool_code | description | quantity
WR001 | Adjustable Wrench 10mm | 5
DR002 | Electric Drill 18V | 3
Edit Tools

Use action buttons to modify tool details. Click edit to update information.

Attendant Management

Create Attendants

Add new tool attendants with unique usernames and shift assignments (A-Z).

Reports & Analytics

10-Day Reports

Generate reports for damaged, spoilt, or missing tools over 10-day periods.

Monthly Reports

Comprehensive monthly audit reports with full system overview.

Important Guidelines

Only administrators can add, edit, or delete tools and attendants
Attendants can only issue tools and view their own records
All exported documents include company logo and branding
